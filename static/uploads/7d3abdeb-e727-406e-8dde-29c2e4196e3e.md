# MindFlow AI - Premium Mind Mapping Application


A professional, keyboard-centric mind mapping application built with vanilla JavaScript, 


CSS (with glassmorphism), and a Python (FastAPI) + MySQL backend.

## ✨ Key Features
- **Keyboard-only Workflow**: Navigate, edit, and create nodes using только hotkeys.
- **Dynamic Themes**: Choose from several professional dark and light themes (Neon, Pro Slate, Deep Forest, Snow White, etc.).
- **Persisted State**: Your mind maps and global user settings (like the active theme) are saved in a MySQL database.
- **Image Support**: Attach, paste, and resize images directly onto nodes.
- **Advanced UI**: Floating resizable nodes, zoomable/pannable canvas, and smooth animations.
- **Capture & Export**: Download your map as a PNG image or view/copy the raw JSON data.

## 🚀 Getting Started

### 1. Prerequisites
- Python 3.8+
- MySQL Server
TEST_CONTENT_BY_AGENT 

### 2. Configuration
Create a `.env` file in the root directory with your MySQL credentials:
```env
DB_HOST=localhost
DB_PORT=3306
DB_USER=your_user
DB_PASSWORD=your_password
DB_NAME=map_db
DB_CHARSET=utf8mb4
```

### 3. Installation
```powershell
# Create and activate virtual environment
python -m venv venv
.\venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### 4. Running the App
```powershell
uvicorn main:app --reload
```
Open `http://localhost:8000` in your browser.
